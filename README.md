# 📘 Akshara-Deepa: AI-Powered SSLC Scholar Matrix

🚀 An AI-powered academic companion designed for Karnataka State Board SSLC students to improve exam preparation using personalized learning, AI evaluation, adaptive quizzes, and real-time analytics.

---

## ✨ Features

🔐 Secure Google Sign-In Authentication  
📊 Real-Time Scholar Dashboard  
🧠 AI-Generated Study Strategies  
📝 Adaptive MCQ & Descriptive Quizzes  
🤖 AI-Based Answer Evaluation  
📈 Subject Performance Analytics  
📡 Radar Strength Visualization  
💬 24/7 AI Tutor Support  
📚 KSEEB Syllabus Hub  
⚡ Real-Time Progress Tracking  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Framer Motion

### Mobile Framework
- Capacitor.js

### Backend & Database
- Firebase Authentication
- Firebase Firestore
- Firebase Analytics

### AI Integration
- Google Gemini 1.5 Flash API

### Charts & Visualization
- Recharts

### Build Tools
- Gradle (API Level 35)

⚙️ Installation
Clone Repository
git clone https://github.com/amruthssss/akshara_deepa.git
cd akshara_deepa
Install Dependencies
npm install
Run Development Server
npm run dev
Build Project
npm run build
Sync Capacitor
npx cap sync
Open Android Studio
npx cap open android
🔥 Firebase Setup

Configure:

Firebase Authentication
Firestore Database
Firebase Analytics
